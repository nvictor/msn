# MSN Changelog

All notable changes to the Melodic Structure Notation (MSN) specification will be documented in this file.

## [1.0.0] - 2026-02-16

### Added
- Initial formalized specification.
- Definition of Motifs, Phrase Roles, and Transformations.
- Support for relative scale-degree notation.
- Conversational semantic roles (?, !, ~, =).
