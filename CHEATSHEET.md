# MSN Cheatsheet (v1.0.0)

## Motifs
- `1` .. `7`: Scale degrees
- `♭` / `♯`: Chromatic alterations
- `↓` / `↑`: Octave shifts

## Phrase Roles
- `?`: Question (Call)
- `!`: Answer (Response)
- `~`: Comment (Reflection)
- `=`: Reinforce (Confirmation)

## Transformations
- `'`: Variation
- `←`: Reverse
- `<`: Expansion
- `>` : Contraction
- `↑` / `↓`: Transpose (Octave)

## Live Mode (Shorthand)
- Use lowercase letters (e.g., `a`, `b`) for motifs.
- Omit brackets `[]` in simple chains if unambiguous.
- Example: `a:135 a? a!~`
